# Deploy to Vercel - Quick Guide

## Option 1: Deploy via Vercel Website (Recommended)

1. **Go to [vercel.com](https://vercel.com)** and sign in
2. Click **"Add New Project"**
3. **Import your Git repository:**
   - If you have GitHub/GitLab/Bitbucket: Connect your repo and import
   - If not: Use the Vercel CLI method below

## Option 2: Deploy via Vercel CLI (Current Setup)

Your project is ready! Run this command in your terminal:

```bash
vercel
```

Then follow the prompts:
- Press Enter to confirm the project settings
- Vercel will auto-detect it's a Vite project
- Your site will be deployed!
